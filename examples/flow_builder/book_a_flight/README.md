### Book a Flight

1. Run `import-all.sh` 
2. Launch the Chat UI with `orchestrate chat start`
3. Pick the `book_a_flight_agent`
4. Type in something like `book a flight from london to the big apple leaving tomorrow returning on Friday for me and my wife`
5. The agentic workflow will guide the user through the flight booking process by presenting a sequence of forms and multi-turn user activities in the chat
